### Para instalar:
- Criando o ambiente isolado e ativando.
```
python3 -m venv .venv
```
```
source .venv/bin/activate
```
- Instalando as dependencias.
```
pip install -r requirements-dev.txt
```

### Verificar a qualidade do codigo.
```
flake8
```
